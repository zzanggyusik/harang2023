# Harang 2023

## 환경 설정
1. 가상환경 생성
   - 터미널 창에 ```python -m venv venv``` 입력
2. 가상환경 실행
   - 윈도우
     - 터미널 창에 ```venv\Scripts\activate``` 입력
   - MAC
     - 터미널 창에 ```source venv\bin\activate``` 입력
3. 패키지 설치(pip)
   1. ```pip install --upgrade pip``` 입력
   2. ```pip install -r requirements.txt``` 입력
